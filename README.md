# open-API-project
 
 This open-API purpose is to view the current temperature, date, as well as the high and low temperatures throughout the week. 

 To open this API project file please do the following: 
 1. Clone the repository. 
 2. Use live server to open HTML file and to run code. 

 [Open API Project](https://github.com/felixnallely/open-API-project)
